## 주제

## 핵심 개념
1. FastAPI는 매우매우 빠르다고 한다
2. uvicorn, swagger, pydantic이 뭔지 대략적으로 알게 되었다.
    - uvicorn: 서버 띄워주는 역할
    - swagger: 대화형 API 문서
        - ReDoc을 이용하면 API 문서화를 대체할 수 있다고 한다
    - pydantic: '런타임에서' 데이터 검증 및 변환
        - <-> mypy: 동적 언어인 Python에서 타입을 강제 -> 대규모 프로젝트로 확장 됐을 때 용이 (코드 작성/컴파일 시점에)
